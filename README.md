## 🚀 Project Overview

Veloura is a premium fine-dining restaurant website designed to deliver a modern, elegant, and interactive user experience. The project simulates a real-world restaurant platform with a digital menu and cart functionality.

---

## 🎤 Project Pitch

**Introduction:**
Veloura is a modern restaurant website built for a premium fine-dining experience in Udupi.

**Problem Statement:**
Many local restaurants lack an engaging and interactive online presence, making it difficult to attract and retain customers digitally.

**Solution:**
This project provides a visually appealing and user-friendly platform with a structured menu, interactive cart system, and modern UI features.

**Impact:**
This solution enhances customer engagement and demonstrates how restaurants can adopt digital platforms for better service and reach.

---

## ✨ Key Features

* 🍽️ Multi-category restaurant menu with images
* 🛒 Add to Cart functionality using LocalStorage
* 🧾 Dynamic cart page with total calculation
* ❌ Remove items and 🗑️ clear cart options
* 🌙 Dark mode toggle
* 🎨 Modern UI with animations and hover effects
* 📱 Responsive and user-friendly design
